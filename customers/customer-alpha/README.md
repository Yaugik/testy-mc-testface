# Customer Alpha browser fixture

Synthetic customer website and browser definitions for the first browser vertical slice.

- `site.yaml` defines deterministic pages, consent behavior, tracking, and a lead form.
- `personas/corporate-visitor.yaml` defines isolated browser settings and session state.
- `fragments/accept-consent.yaml` demonstrates reusable journey steps.
- `journeys/lead-capture.yaml` exercises navigation, form interaction, stable selectors, and artifact policy.

All domains, addresses, and identities are synthetic.
